# Quick start: Next.js scaffold

This repository now includes a minimal Next.js scaffold (app router).

Prerequisites
- Node.js 18+ is recommended.

Run locally (PowerShell)

```powershell
cd "c:\Users\mattm\OneDrive\Documents\github-repositories\HoosGotTime"
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

Notes
- If you prefer TypeScript, run `npx next telemetry disable` and convert files to .tsx and add tsconfig.
- To publish, run `npm run build` and `npm run start` on the server.
